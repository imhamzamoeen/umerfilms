# Story 3.2: Work With Me Page with Services List

## Status
Draft

## Story
**As a** visitor,
**I want** to see what services Umer offers,
**so that** I can understand how he can help with my project.

## Acceptance Criteria

1. Work With Me page is accessible at `/contact` route (or `/work-with-me`)
2. Page displays heading ("Work With Me" or "Let's Create Together")
3. Brief intro paragraph sets expectations for working with Umer
4. Services section lists available service categories with descriptions:
   - Commercial/Brand Content
   - Music Videos
   - Wedding/Event Coverage
   - Short Films/Creative Projects
   - (Exact services TBD - use placeholders)
5. Each service displays: title, brief description (1-2 sentences)
6. "Get Quote" messaging is clear (no public pricing)
7. Page layout has services section above contact form section
8. Section heading for contact form: "Get in Touch" or "Request a Quote"
9. Page is responsive with appropriate spacing
10. Page metadata (title, description) is set for SEO

// src/app/contact/page.tsx

import { Metadata } from 'next';

  },
];

export default function ContactPage() {
  return (
    <PageWrapper>
      <div className="py-16 md:py-20">
        {/* Page Heading */}
        <h1 className="text-4xl md:text-5xl font-bold mb-6 text-center">
          Let's Create Together
        </h1>

        {/* Intro Paragraph */}
        <p className="text-lg md:text-xl text-gray-300 text-center max-w-3xl mx-auto mb-16">
          Every project is unique, and I provide custom quotes tailored to your vision
          and goals. Share your project details below, and I'll get back to you within
          24-48 hours to discuss how we can bring your ideas to life.
        </p>

        {/* Services Section */}
        <div className="mb-16 md:mb-20">
          <h2 className="text-3xl md:text-4xl font-bold mb-8 md:mb-12 text-center">
            Services
          </h2>

          <div className="grid grid-cols-1 md:grid-cols-2 gap-6 md:gap-8 max-w-4xl mx-auto">

              <div
                key={service.title}
                className="bg-gray-800 rounded-lg p-6 md:p-8"
              >
                <h3 className="text-xl md:text-2xl font-semibold mb-3">
                  {service.title}
                </h3>
                <p className="text-gray-300 leading-relaxed">
                  {service.description}
                </p>
              </div>
            ))}
          </div>
        </div>

        {/* Contact Form Section */}

          <h2 className="text-3xl md:text-4xl font-bold mb-8 md:mb-12 text-center">

          {/* <ContactForm /> */}
          <div className="max-w-2xl mx-auto bg-gray-800 rounded-lg p-8 text-center text-gray-400">
            Contact form coming soon (Story 3.3)

**Option 1: Grid Cards (Recommended):**
```typescript

    </div>
  ))}
</div>
```
- Clean 2-column layout on desktop
- Stacks to 1 column on mobile
- Card design with subtle background

**Option 2: Simple List:**
```typescript
<div className="space-y-8">
  {services.map((service) => (
    <div key={service.title}>
      <h3 className="text-2xl font-semibold mb-2">{service.title}</h3>
      <p className="text-gray-300">{service.description}</p>
    </div>
  ))}
</div>
```
- Simpler, more minimal
- Single column on all screens

**Recommendation:** Option 1 (Grid Cards) - more visual interest.

**Test file location:**

**Required test patterns:**

```typescript
// src/app/contact/page.test.tsx

import { render, screen } from '@testing-library/react';
import { describe, it, expect } from 'vitest';
import ContactPage from './page';

describe('ContactPage', () => {
  it('renders page heading', () => {
    render(<ContactPage />);
    expect(screen.getByRole('heading', { level: 1 })).toHaveTextContent(/work with me|let's create together/i);
  });

  it('renders intro paragraph', () => {
    render(<ContactPage />);
    expect(screen.getByText(/custom quote|project/i)).toBeInTheDocument();
  });

  it('renders services section heading', () => {
    render(<ContactPage />);
    expect(screen.getByRole('heading', { name: /services/i })).toBeInTheDocument();
  });

  it('renders all 4 service categories', () => {
    render(<ContactPage />);
    expect(screen.getByText(/commercial.*brand content/i)).toBeInTheDocument();
    expect(screen.getByText(/music video/i)).toBeInTheDocument();
    expect(screen.getByText(/wedding.*event/i)).toBeInTheDocument();
    expect(screen.getByText(/short film/i)).toBeInTheDocument();
  });

  it('renders contact form section heading', () => {
    render(<ContactPage />);
    expect(screen.getByRole('heading', { name: /get in touch|request a quote/i })).toBeInTheDocument();
  });

  it('applies responsive grid classes to services', () => {
    const { container } = render(<ContactPage />);
    const grid = container.querySelector('.grid');
    expect(grid).toHaveClass('grid-cols-1');
    expect(grid).toHaveClass('md:grid-cols-2');
  });

  it('includes get quote messaging', () => {
    render(<ContactPage />);
    expect(screen.getByText(/custom quote|pricing|request/i)).toBeInTheDocument();
  });
});
```
